# 🍔 Food Delivery App

A modern and user-friendly Food Delivery Application developed using Python. The application allows users to browse food items, place orders, calculate bills, and manage food delivery operations efficiently.

![Python](https://img.shields.io/badge/Python-3.10-blue)
![Food Delivery](https://img.shields.io/badge/Food-Delivery-orange)
![Gradio](https://img.shields.io/badge/Gradio-Web%20UI-green)


---

# 📖 Overview

The Food Delivery App provides a seamless platform for customers to order food online. Users can select food items from the menu, specify quantities, and instantly receive the total bill. The system simplifies food ordering and serves as a foundation for building advanced online food delivery solutions.

---

# ✨ Features

- 🍕 Browse Food Menu
- 🛒 Place Food Orders
- 🔢 Select Quantity
- 💰 Automatic Bill Calculation
- 📋 Order Summary
- ⚡ Real-Time Processing
- 🎨 User-Friendly Interface

# 📊 How It Works

1. User selects a food item.
2. User enters quantity.
3. System calculates total amount.
4. Order details are displayed.
5. User receives order confirmation.
